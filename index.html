Example Work

CV Website

I built this CV website. I will expand upon tech stack and hosting arrangements, but firstly some important points to note:

- It was built using mobile first design principles
- Subsequently, it is responsive for different screen sizes
- It is under source control (Git).
- You can view the GitHub repository here.

This website is a static site, built purely in HTML, CSS and JS. For CSS I have included the Bootstrap library and most of the styling is derived from it, but there are some custom stylings sprinkled about. For JS I have used a little JQuery to help display the mobile navigation. I opted for JQuery as it is already bundled with Bootstrap so there was no extra overhead. I could also achieve the same effect using vanilla JS.

This site is hosted on a dedicated server running Ubuntu 20.04. I have opted for Apache as my HTTP server. As the dedicated server has 1000mb/s upload/download speed, it should be lighting fast in most instances.

Indrani Bridal

Indrani Bridal is a wedding dress shop which I worked out during 2020. During that time, I built the company’s website which can be viewed here.

The backend is built in C# using .net Framework. The DB is MSSQl, and it was built using Entity Framework (code first). The frontend makes use of several different libraries, Bootstrap, AoS and JQuery to name a few.

It is hosted in Azure on one of their virtual machines. The SQL is hosted on an Azure SQL Server. As the account is on a free tier, the loading time is not as fast as I would like. Once the budget is there I am going to migrate away from Azure and towards an Ubuntu dedicated server.

News Watch

News Watch(NW) is a news indexing service which scrapes news articles and indexes the results. You can see its web interface here. The web interface shows the results of all the news articles indexed. You can also set the scrapers search parameters and see its logs via this site. They are hidden behind user authentication so won't be available to a standard user. I would be happy to show the GitHub repository and show the different parts of the website to any recruiters who may be interested. It's important to note though, that this project is still a work in progress.

NW consists of two different programs. Program 1 is the news scraper and program 2 is the website; both of which are built in C# using .net Core. The search results are indexed via MSSQL. Both programs communicate with the DB via Entity Framework and the DB was built code first.

It is hosted on a dedicated server running Ubuntu 20.04. The HTTP server is Apache. The site itself is hosted on a local Kestrel server, but Apache does a proxy forward to the local Kestrel server in order to serve the website to external guests.
