Previous Projects Completed

Example Work

CV Website

This CV website you are currently looking at was, needless to say, built by me. The website is built with mobile first principles and is responsive for different screen sizes. It is under source control (Git) and you can view the github depository here. To give you more insight I will expand upon tech stack used and hosting arrangments in the following paragraphs.

Firstly, tech stack. This website is a static site, built purely in HTML, CSS and JS. For CSS I have included the Bootstrap library and the vast majority of the styling are dervied from it, but there are some custom stylings sprinkled about. For JS I have used a little JQuery to help display the mobile navigation. I opted for JQuery as it is already bundled with bootstrap so there was no extra overhead. I could also acheive the same effect using vanilla JS.

Secondly, hosting arrangements. This site is hosted on a dedicated server running Ubuntu 20.04. I have opted for Apache as my HTTP server. As a dedicated server with 1000mb/s upload/download speed, it should be lighting fast in most instances.

Indrani Bridal

Indrani Bridal is a wedding dress shop which I worked out during 2020. During that time I built the companies website which can be viewed here.

The backend is built in c# using .net framework. The DB is MSSQl, and it was built using entity framework (code first). The frontend makes use of a number of different libraries, Bootstrap, AoS and JQuery to name a few.

It is hosted in Azure on one of their virtual machines. The SQL is hosted on an Azure SQL server. As the account is free tier, the loading time is not as fast as I would like. Once the budget is there I am going to migrate away from Azure and towards a Ubuntu dedicated server.

News Watch

News Watch(NW) is a news indexing service which scrapes news articles and indexes the results. You can see its web interface here. The web interface shows the results of all the news articles indexed. You can also set all of the scrapers search parameters and see its logs via the site. This is hidden behind an identity role so wont be available to a standard user. I would be happy to show the GitHub repository and show the different parts of the website to any recruiters who maybe interested.

NW consists of two different programs. Program 1 is the news scraper and program 2 is the website; both of which are built in C# using .net core. The search results are indexed via MSSQL. Both programs communicate with the DB via Entity Framework and the DB was built code first.

It is hosted on a dedicated server running Ubuntu 20.04. The HTTP server is Apache. The site itself is hosted on a Kestrel server, but Apache does a proxy forward to the local Kestrel server in order to serve the website to external guests.
